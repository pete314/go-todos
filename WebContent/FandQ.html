Task Manager

1. How do i use this application?

Once logged into Task Manager all the options the the user needs are seen as icons in the menu bar. These options include, manage tasks, profile settings, this FandQ page and logout.

2. How do i use the Manage Tasks Feature?

The manage tasks feature contains a table displaying each one of the users task and which ones are currently important, done or not done using different colours. Beside each task is a checkbox where the user can select one or multiple tasks. Once tasks have been selected a user can then choose to finish, edit or delete a particular task.

3. What can i do in the profile settings?

The profile settings allow a user to change his/her password, email and name. It also allows the user to disable or enable notification settings that alert a user when a task is coming up.
